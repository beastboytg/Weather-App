# 🌤️ Weather Forecast Web App

A full-stack weather application built using **Node.js, Express.js, EJS, HTML, CSS, and JavaScript**. It fetches real-time weather data using the WeatherAPI and displays a dynamic, animated UI based on current weather conditions.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature and conditions
- 📅 3-day weather forecast
- 🌅 Sunrise and 🌇 sunset timings
- 💨 Wind speed, humidity, UV index
- 🌧️ Dynamic weather animations (rain, snow, fog, thunder)
- 🌗 Day/Night theme switching
- ⚠️ Error handling for invalid cities
- 🎨 Smooth UI transitions and skeleton loading effects

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, EJS
- **Backend:** Node.js, Express.js
- **API:** WeatherAPI
- **Other:** Axios, dotenv

## 📦 Installation & Setup

```Create your api key
  https://www.weatherapi.com
```

### 1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

### 2. Navigate into the project folder

```bash
cd weather-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Create a .env file in the root directory

```bash
WEATHER_API_KEY=your_api_key_here
```

### 5. Start the server

```bash
node index.js
```

### 6. Open in browser

```bash
http://localhost:3000
```
